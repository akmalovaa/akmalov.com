# Library

Сборник интересных библиотек и фреймворков на Python

## Веб-фреймворки
- [**FastAPI**](https://github.com/tiangolo/fastapi) — современный, быстрый веб-фреймворк для создания API с автоматической документацией
- [**Litestar**](https://github.com/litestar-org/litestar) — высокопроизводительный ASGI веб-фреймворк для создания REST API и веб-приложений
- [**django-modern-rest**](https://github.com/wemake-services/django-modern-rest) — набор лучших практик и инструментов для создания современных REST API на Django

## Базы данных и ORM
- [**SQLAlchemy**](https://github.com/sqlalchemy/sqlalchemy) — мощная ORM и SQL toolkit для работы с реляционными БД
- [**asyncpg**](https://github.com/MagicStack/asyncpg) — самый быстрый асинхронный драйвер PostgreSQL
- [**redis-py**](https://github.com/redis/redis-py) — клиент для работы с Redis (кэширование, очереди, pub/sub)

## Асинхронное и очереди
- [**aiohttp**](https://github.com/aio-libs/aiohttp) — асинхронный HTTP клиент/сервер для высоконагруженных приложений
- [**aiokafka**](https://github.com/aio-libs/aiokafka) — асинхронный клиент Apache Kafka для обработки потоков данных
- [**aio-pika**](https://github.com/mosquito/aio-pika) — асинхронная библиотека для работы с RabbitMQ

## HTTP клиенты
- [**httpx**](https://github.com/encode/httpx) — современный HTTP клиент с поддержкой sync/async и HTTP/2
- [**requests**](https://github.com/psf/requests) — классический и самый популярный HTTP клиент

## Валидация и сериализация
- [**Pydantic**](https://github.com/pydantic/pydantic) — валидация данных через аннотации типов (используется в FastAPI)

## Мониторинг и observability
- [**prometheus-client**](https://github.com/prometheus/client_python) — экспорт метрик в Prometheus для мониторинга
- [**sentry-sdk**](https://github.com/getsentry/sentry-python) — отслеживание ошибок и производительности в production
- [**structlog**](https://github.com/hynek/structlog) — структурированное логирование для удобного анализа логов
- [**opentelemetry**](https://github.com/open-telemetry/opentelemetry-python) — распределенная трассировка и сбор метрик

## Обработка данных
- [**Pandas**](https://github.com/pandas-dev/pandas) — анализ и обработка данных (стандарт индустрии)
- [**Polars**](https://github.com/pola-rs/polars) — современная и очень быстрая альтернатива Pandas на Rust
- [**NumPy**](https://github.com/numpy/numpy) — фундаментальная библиотека для научных вычислений

## Тестирование
- [**pytest**](https://github.com/pytest-dev/pytest) — современный фреймворк для тестирования
- [**pytest-asyncio**](https://github.com/pytest-dev/pytest-asyncio) — поддержка async тестов
- [**hypothesis**](https://github.com/HypothesisWorks/hypothesis) — property-based тестирование

## Боты и интеграции
- [**Aiogram**](https://github.com/aiogram/aiogram) — асинхронная библиотека для разработки Telegram ботов на основе Bot API
- [**Python-gitlab**](https://github.com/python-gitlab/python-gitlab) — Python библиотека для работы с GitLab API
- [**RouterOS-api**](https://github.com/socialwifi/RouterOS-api) — библиотека для работы с API устройств RouterOS (MikroTik)
