Homelab

Homelab - интересные проекты и сервисы

Кластеры​

  • Proxmox VE
  • Kubernetes (k0s)

Базовые проекты​

  • Homepage - Стартовая страница (дашборд) с часто используемыми ссылками
  • Kaneo - Канбан доска для задач
  • Jellyfin - Медиа-сервер
  • Vaultwarden - Менеджер паролей
  • Immich - Аналог гугл фото, хранение и обмен фотографиями
  • Paperless-ngx - Система управления документами
  • Home Assistant - Автоматизация дома
  • Ollama - Локальный LLM
  • Open Web UI - Веб интерфейс для LLM
  • n8n - Автоматизация задач
  • RustDesk - Сервер для удалённого доступа аналог TeamViewer, AnyDesk
  • Mailcow - Почтовый сервер
  • Stirling-PDF - Работа с PDF файлами
  • Reveal.js - Создание HTML презентаций
  • Linkding или Karakeep - Управление закладками нужно сравнить пока изучаю

Инфраструктура​

  • Pangolin - Организация внешнего доступа
  • Traefik - Удобный реверс-прокси
  • AdGuard Home - DNS сервер с блокировкой рекламы
  • Grafana - Визуализация метрик
  • Prometheus - Сбор метрик
  • Node Exporter - экспортер метрик для узлов
  • Alertmanager - Управление оповещениями
  • Loki - Для сбора и анализа логов
  • Tempo - Для просмотра трассировок
  • Uptime Kuma - Мониторинг доступности
  • CrowdSec - Блокировка подозрительных IP адресов
  • Speedtest-rust - Тестирование пропускной способности канала
  • PostgreSQL - База данных
  • Minio S3 - Хранилище объектов
  • Duplicati - Резервное копирование
  • Syncthing - Синхронизация файлов
  • Cockpit - Веб интерфейс для управления сервером
  • UpSnap - Wake on LAN через веб интерфейс

Разработка​

  • Gitea - Свое Git хранилище
  • Sentry - Отслеживание ошибок
  • Centrifugo - Вебсокет сервер
  • Valkey - база данных аналог Redis
  • MeiliSearch - Поисковый движок
  • IT-tools - Набор инструментов помогает в разработке
  • Nocodb - Работа с базами в веб интерфейсе + API
  • Penpot - OpenSource аналог Figma
  • Hoppscotch - Аналог Postman для тестирования API

K8s​

  • External Secret Operator - Управление секретами из внешних хранилищ
  • Cert Manager - Автоматическое управление TLS сертификатами
  • NFS- Динамическое выделение хранилища через NFS
  • HashiCorp Vault - Хранилище секретов
  • Kube-virt - Виртуальные машины в Kubernetes

Игровые серверы​

  • Satisfactory - Строительство заводов
  • Enshrouded - Песочница с элементами выживания
  • CS2 - cs2
